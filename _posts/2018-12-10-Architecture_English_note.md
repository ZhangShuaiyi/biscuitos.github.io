# O

##### <span id="occupy">occupy</span>

> 占用

The user space **occupies** the lower portion of the address space, starting
from address 0 and extending up to the TASK_SIZE.

用户空间占用了低地址部分，该部分从 0 开始一直拓展到 TASK_SIZE 处。

{% highlight ruby %}
0                                               TASK_SIZE            4G
+-----------------------------------------------+---------------------+
|                                               |                     |
|                                               |                     |
|                                               |                     |
+-----------------------------------------------+---------------------+
| <---------------- User Space ---------------->|
{% endhighlight %}
