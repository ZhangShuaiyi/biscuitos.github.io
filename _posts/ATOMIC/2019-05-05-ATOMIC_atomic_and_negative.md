# <span id="源码分析">源码分析</span>

{% highlight ruby %}
/**
 * atomic_add_negative - add and test if negative
 * @i: integer value to add
 * @v: pointer of type atomic_t
 *
 * Atomically adds @i to @v and returns true
 * if the result is negative, or false when
 * result is greater than or equal to zero.
 */
#ifndef atomic_add_negative
static inline bool atomic_add_negative(int i, atomic_t *v)
{
        return atomic_add_return(i, v) < 0;
}
#endif
{% endhighlight %}

atomic_add_negative() 函数对 atomic_t 变量进行加法，并判断结果是否为负数，如果
是负数则返回 true。参数 i 表示要增加的数量；v 指向 atomic_t 变量。函数直接调用
atomic_add_return() 函数对 atomic_t 变量进行加法，然后直接判断节点是否小于 0。

/* atomic_* */
static __init int atomic_demo_init(void)
{
	/* Legacy data */
	printk("[0]Atomic: %d\n", atomic_read(&BiscuitOS_counter));

	if (atomic_add_negative(2, &BiscuitOS_counter))
		printk("Atomic is negative\n");
	else
		printk("Atomic is no negative\n");

	return 0;
}

#### <span id="驱动安装">驱动安装</span>

驱动的安装很简单，首先将驱动放到 drivers/BiscuitOS/ 目录下，命名为 atomic.c，
然后修改 Kconfig 文件，添加内容参考如下：

{% highlight bash %}
diff --git a/drivers/BiscuitOS/Kconfig b/drivers/BiscuitOS/Kconfig
index 4edc5a5..1a9abee 100644
--- a/drivers/BiscuitOS/Kconfig
+++ b/drivers/BiscuitOS/Kconfig
@@ -6,4 +6,14 @@ if BISCUITOS_DRV
config BISCUITOS_MISC
        bool "BiscuitOS misc driver"
+config BISCUITOS_ATOMIC
+       bool "atomic"
+
+if BISCUITOS_ATOMIC
+
+config DEBUG_BISCUITOS_ATOMIC
+       bool "atomic_add_negative"
+
+endif # BISCUITOS_ATOMIC
+
endif # BISCUITOS_DRV
{% endhighlight %}

接着修改 Makefile，请参考如下修改：

{% highlight bash %}
diff --git a/drivers/BiscuitOS/Makefile b/drivers/BiscuitOS/Makefile
index 82004c9..9909149 100644
--- a/drivers/BiscuitOS/Makefile
+++ b/drivers/BiscuitOS/Makefile
@@ -1 +1,2 @@
obj-$(CONFIG_BISCUITOS_MISC)     += BiscuitOS_drv.o
+obj-$(CONFIG_BISCUITOS_ATOMIC)  += atomic.o
--
{% endhighlight %}
